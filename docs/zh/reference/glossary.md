# 术语表

## Agent Knowledge

面向 AI Agent 的有来源知识包标准。

## Knowledge pack

包含 `KNOWLEDGE.md` 以及可选 sources、wiki、compiled、indexes、runs、assets 的目录。

## Agent Skill

告诉 Agent 如何执行任务的流程包，入口是 `SKILL.md`。

## Source

原始证据，如文档、访谈、网页、PDF、会议记录。

## Wiki

从 sources 整理出的结构化长期知识。

## Compiled view

运行时紧凑视图，如 `compiled/facts.md`。

## Index

可重建检索工件，如全文、向量或图索引。

## Context resolver

选择哪些知识进入模型上下文的客户端组件。
