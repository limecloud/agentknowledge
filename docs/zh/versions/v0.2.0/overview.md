# v0.2.0 概览

0.2.0 是首个公开版本之后的高密度草案。

它补齐了三个原来缺失的关键层：

1. **发现** - description 如何帮助知识包被正确选择。
2. **维护** - 如何用脚本、Skills 和工具更新知识，而不把知识压成指令。
3. **评估** - 如何持续衡量发现、溯源和回答质量。

同时，它扩展了 Karpathy LLM Wiki 参考、客户端接入说明和版本快照。

## 亮点

- 更清晰的 Skills 与 Knowledge 边界。
- 明确的 description 优化指导。
- 知识包发现评估。
- 维护自动化指导。
- 客户端信任、激活和受保护上下文规则。
- 完整的 LLM Wiki 模式映射。
- 文档站复制 Markdown 支持。
- 多语言文档和版本快照。
