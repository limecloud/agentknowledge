# Personal IP pack

```text
lilei-personal-ip/
├── KNOWLEDGE.md
├── sources/
│   ├── interview.md
│   └── public-posts.md
├── wiki/
│   ├── entities/lilei.md
│   ├── concepts/content-method.md
│   └── open-questions/index.md
└── compiled/
    ├── facts.md
    ├── voice.md
    ├── stories.md
    └── boundaries.md
```

## Use cases

- founder introduction
- short video scripts
- event speech opening
- profile rewrite
- sales conversation style guide

## Boundary examples

- Do not invent client names or revenue numbers.
- Mark missing achievements as unknown.
- Keep voice grounded in confirmed interviews and public writing.
