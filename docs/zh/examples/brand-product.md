# 品牌产品知识包

```text
acme-widget/
├── KNOWLEDGE.md
├── sources/
│   ├── product-one-pager.md
│   ├── pricing.md
│   └── compliance.md
└── compiled/
    ├── facts.md
    ├── voice.md
    ├── playbook.md
    └── boundaries.md
```

## 适用场景

- 产品详情页
- 销售邮件
- 社交媒体文案
- 合作伙伴简报
- 客服回复

## 边界示例

- 没有来源时不写医疗、金融或监管声明。
- 不编造价格和库存。
- 优先使用 `compiled/voice.md` 中的定位表达。
